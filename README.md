ytoj2me
=======

圆通客户端j2me版

1st.rar、2nd.rar、3rd.rar分别是本软件的前三个版本

开发设备是Nokia 5500 Sports，运行截图在Screenshot文件夹内

作者：宝哥哥

应用环境：支持j2me的手机设备

开发环境：Java/Eclipse

完成时间：2009年5月31日
